Sobre

O LucasFlix é uma plataforma de filmes e séries desenvolvida como projeto do primeiro semestre do curso de Análise e Desenvolvimento de Sistemas.

O site permite navegar por um catálogo de filmes, visualizar um slider com os títulos em destaque e acessar informações detalhadas de cada filme.

Todo o conteúdo é gerado dinamicamente via JavaScript a partir de uma estrutura de dados JSON.
